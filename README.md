# World Center 📡
**Where Words Become Worlds.**

Founded by Adesina Israel, **World Center** is a complete podcasting command center designed to inspire the next generation through teaching and motivational content.

## 🎙️ Features
* **Podcast Studio:** Integrated production workflow from topic ideation to publishing.
* **Heavy Crowns Hub:** Dedicated section for tracking live episodes and metrics.
* **Educational Library:** Access to a curated collection of authors and wisdom.
* **Multi-Language Support:** Tools to manage and translate content across 5+ languages.
* **Research Suite:** Built-in Dictionary and Wikipedia widgets for real-time script research.

## 🛠️ Technology Stack
* **Frontend:** HTML5, CSS3 (Syne & Cormorant Garamond fonts)
* **Design:** Premium dark-mode interface with gold accents (#E8B84B)
* **Interactivity:** Vanilla JavaScript for page switching and tool functionality
